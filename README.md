# Lavalampe

This project is a fork of the original watchface created by [jmlaitpebble](https://github.com/jmlaitpebble) in 2014.

The goal of this fork is to improve readability by displaying the corresponding power of two on each blob, for example 2¹, 2², 2³, and so on. In the original version, the meaning of each blob is not clearly indicated.

The colors should also be customizable through the watchface settings.

This fork is called **Lavalamp**.
